# 专精特新企业申报助手 (SRDI Application)

基于**2026年工信部最新政策**（工信部企业〔2026〕2号）的专精特新企业申报材料智能生成技能。

## 功能亮点

- **三梯度全覆盖**: 创新型中小企业 → 省级专精特新 → 国家级小巨人
- **智能资格评估**: 自动逐项检查硬性指标，标注达标/未达标/边界状态
- **全套材料生成**: 申请书、自查清单、佐证材料清单、市场地位说明、研发费用说明
- **交互式HTML报告**: 六维雷达图、Tab切换、可打印、可直接提交
- **避坑指南**: 2026新政10条红线 + 常见失败原因 + 准备建议
- **政策红利**: 财政奖励、税收优惠、融资支持、政府采购、上市辅导

## 使用方式

```
帮我生成专精特新小巨人申报材料，我们公司年营收8000万，有6项发明专利
```

## 技术栈

Python 3.x | Chart.js | 纯离线可用

## 输出示例

- 📊 申报概览仪表盘（达标率+六维雷达图）
- 📋 硬性指标逐项检查表
- 📝 完整申请书（八大部分）
- 📎 佐证材料待办清单（带勾选框）
- ⚠️ 避坑指南（2026新政红线）
- 💰 政策红利说明
