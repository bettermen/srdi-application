# 专精特新企业申报材料智能生成技能

## 概述

基于2026年工信部最新《优质中小企业梯度培育管理办法》（工信部企业〔2026〕2号），为企业提供三个梯度（创新型中小企业 → 省级专精特新中小企业 → 国家级专精特新"小巨人"）的全套申报材料智能生成服务。

## 三个梯度体系

### 梯度一：创新型中小企业（基础层）
- 前置条件：境内注册、独立法人
- 评分要求：创新能力/成长性/专业化三类指标综合评价≥60分
- 直通条件：近三年获国家级/省部级科技奖励，高新技术企业，国家级技术创新示范企业，省部级以上研发机构，近三年新增股权融资≥500万

### 梯度二：省级专精特新中小企业（中间层）
- 前置资格：已获得创新型中小企业称号
- 从事特定细分市场≥3年
- 营收≥1500万 或 近2年股权融资≥2000万
- 主营业务占比≥80%，资产负债率≤80%
- 近两年研发费用每年≥100万，占营收≥3%
- I类知识产权≥1项（自主申请）
- 发展评价得分≥50分

### 梯度三：国家级专精特新"小巨人"（塔尖层）
- 前置资格：已获得省级专精特新中小企业称号
- 成立≥2年，深耕细分行业≥3年
- 营收≥5000万，主营业务占比≥90%
- 近两年营收复合增长率≥5%（复核免此项）
- 资产负债率≤75%
- 近两年研发费用合计≥1200万，每年占营收≥3%
- I类知识产权≥4项（自主申请，与主营直接相关）
- 主导产品市占率≥10%或国内前三
- 发展评价得分≥60分（2026新增）
- 属于六基领域/制造强国十大产业/强链补链环节

## 工作流程

### 阶段一：需求确认
根据用户输入判断目标梯度：
- 用户说"创新型"/"创新中小企业" → 梯度一
- 用户说"专精特新"/"省级" → 梯度二
- 用户说"小巨人"/"国家级" → 梯度三
- 用户未明确 → 询问目标梯度

### 阶段二：企业信息采集
根据目标梯度，采集以下信息：
1. 企业基本信息：名称、注册地、成立时间、所属行业
2. 财务数据：近两年营收、主营收入、研发费用、资产负债率
3. 知识产权：I类专利数量、专利名称列表、是否自主申请
4. 市场地位：主导产品、细分市场、市占率估算
5. 资质证书：已有认定证书
6. 合规情况：近三年是否有重大安全/环保/税务事故

### 阶段三：资格预评估
基于2026最新标准进行硬性指标检查：
- 逐项对照认定条件，标注✅达标/❌未达标/⚠️边界
- 给出综合判定：建议申报 / 补强后再报 / 暂不满足
- 针对未达标项给出具体补强建议

### 阶段四：材料生成
根据目标梯度生成全套材料：

**通用材料清单：**
1. 申请书正文（按官方模板格式）
2. 企业基本情况和主导产品说明
3. 财务数据汇总表（营收/研发/负债率）
4. 知识产权清单及说明
5. 研发费用专项说明
6. 市场地位与竞争力说明
7. 合规经营承诺书
8. 自查评分表
9. 佐证材料清单（待办清单）

**梯度二额外材料：**
- 股权融资证明说明（如适用）

**梯度三额外材料：**
- 产业链关键环节说明
- 近两年营收复合增长率计算
- 专精特新发展评价得分预估

### 阶段五：报告输出
生成交互式HTML可视化报告，包含：
- 申报概览仪表盘（雷达图+达标仪表）
- 逐项指标对比表
- 完整申请书（可复制/打印）
- 材料待办清单（带勾选框）
- 避坑指南（2026最新红线）
- 政策红利说明

## 关键政策规则（2026版）

### 硬性红线（一票否决）
1. 外购/转让/受让专利不计入I类知识产权
2. 营收<5000万工信部不受理小巨人
3. 主营占比<90%直接淘汰（小巨人）
4. 资产负债率>75%一票否决（小巨人）
5. 数据造假→取消资格+3年禁止申报+追缴补贴+失信公示
6. 必须先在省级专精特新，不可越级申报小巨人

### 2026新政要点
1. 材料大幅减负：无需第三方市占率证明、无需纸质专利证书
2. 审计报告必须是财政部监管平台已赋码电子原件
3. 专利、市占由工信部大数据官方核验
4. AI+大数据审核，双随机抽查
5. 全国统一标准，取消地方特色指标
6. 新增60分综合评价门槛（小巨人）

## 使用说明

### 输入方式
1. **快速模式**：一句话描述企业情况和目标梯度
   - 例："帮我生成专精特新小巨人申报材料，我们公司年营收8000万，有6项发明专利"
2. **引导模式**：只说目标，技能会逐步引导填写关键信息
   - 例："我要申报省级专精特新"
3. **完整模式**：提供详细企业数据文件（CSV/JSON）

### 输出物
- 交互式HTML可视化报告（主输出）
- 申请书纯文本版（可复制到申报系统）
- 佐证材料待办清单

## 注意事项
1. 生成的材料仅供参考，正式提交前请仔细核对并咨询专业机构
2. 政策随时可能调整，以工信部官网最新通知为准

4. 所有数据必须真实准确，造假后果严重
